# Contributing

Thanks for taking interest in this list!

## How to add a resource

1. Ensure the resource is Python-focused and relevant to TouchDesigner.
2. Add a single bullet entry in the appropriate section of `README.md`.
3. If the entry is a curated collection (not a single tool/tutorial), add a summary in `curations/owner-repo.md`.
4. Open a pull request.

## Format

```
- [resource-name](url) - Brief description of what it is and why it matters.
```

Use regular dashes (not em-dashes) between the link and description. Keep descriptions concise. See existing entries for style reference.
