# dylanroscover/TDComponents

**Stars:** 98 | **URL:** https://github.com/dylanroscover/TDComponents | **Language:** Python

## Overview

A whimsical collection of polished, reusable TouchDesigner components with heavy Python focus.

## Components

| Component | Description |
|-----------|-------------|
| **Chop Recorder** | Record multi-channel CHOPs to disk in real-time, play back later. Ideal for LiDAR/MIDI data. |
| **Color Curves** | Graphical curve-based per-channel color grading (Master/R/G/B). Presets, histogram, split-view compare. |
| **Comper** | Texel-based GLSL compositor with per-layer pos/rot/scale/matte. Sequence parameter control. |
| **Hexler** | RGB-to-Hex color conversion utility. |
| **Optimeister** | Conditional cooking: only cooks CHOPs (Delay/Filter/Lag/Logic/Timer/Trigger/Speed/Slope) when data changes. |
| **Masker** | UI-based mask creation with multiple objects, surfaces, presets, curve/invert support. |
| **Playback** | Video player with geometry COMP, crossfade, scrub, loop/oneshot. |
| **Step N Repeat** | GLSL brick/grid pattern tiler for logo+background compositing. |
| **Timecode** | Fast texture-slicing-based timecode renderer (2x faster than Text TOP). |

## Python Usage

All components are Python-driven with well-documented parameter interfaces, CHOP outputs, and expression-based bindings.

## Relevance

Most Python-heavy curated collection. Actively maintained (Feb 2026). Production-quality components.
