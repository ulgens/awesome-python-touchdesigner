# riebschlager/touchdesigner-playground

**Stars:** 90 | **URL:** https://github.com/riebschlager/touchdesigner-playground | **Language:** Mostly networks

## Overview

A learning journal of 30+ TouchDesigner sketches by riebschlager. Each folder has a `.toe` file and thumbnail preview.

## Notable Sketches

- **ball-grid** — Line SOP + Spring SOP instanced grid fabric simulation
- **brush-ribbon** — 3D paintbrush stroke instancing
- **cloud-walker** — Noise-driven height map phong material
- **eyeballs** — PBR texture fun with instanced spheres
- **face-mixing** — Noise mattes for collage-style layer compositing
- **fire-ball** — Particle system sphere blend fire/smoke
- **json-from-api** — REST API JSON data binding example
- **pour-painting** — Noise + cache-selecting acrylic pour imitation
- **spiky-ouroboros** — Noise-driven instanced geometry snake
- **vortex** — Noise-affected circles through rectangle layers

## Python Usage

Minimal. Mostly SOP/TOP networks with some Python for data binding and parameter control.

## Relevance

Great for learning TD techniques and understanding how to build generative visuals. Not production-optimized. Last updated Dec 2020.
