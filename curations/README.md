## 1. danielsamson/awesome-touchdesigner

- **Stars:** 22
- **URL:** https://github.com/danielsamson/awesome-touchdesigner
- **Python focus:** No
- **Summary:** Early curated list of TouchDesigner resources covering books (An Introduction to TouchDesigner, Multimedia Programming Using Max/MSP and TouchDesigner), free/paid video tutorials (Summit Berlin 2018, Lynda.com, Matthew Ragan), companies (Obscura Digital, Moment Factory), and events (TouchDesigner Summit 2019). Inspired by Awesome Creative Coding. Lightweight and beginner-oriented.
- **Last updated:** Mar 2019

---

## 2. dylanroscover/TDComponents

- **Stars:** 98
- **URL:** https://github.com/dylanroscover/TDComponents
- **Python focus:** Yes (heavily Python-driven)
- **Summary:** A whimsical collection of polished, reusable .tox components for TouchDesigner. Includes Chop Recorder (multi-channel CHOP recording/playback), Color Curves (graphical curve editor), Comper (texel-based GLSL compositor), Optimeister (conditional cooking for CHOPs), Masker (UI-based mask creation), Playback (video player with crossfade), Step N Repeat (brick/grid pattern tiler), and Timecode (fast texture-based timecode renderer). Strong Python emphasis with practical, production-ready tools.
- **Last updated:** Feb 2026

---

## 3. L05/TouchDesigner

- **Stars:** 65
- **URL:** https://github.com/L05/TouchDesigner
- **Python focus:** Minimal (primarily component-based)
- **Summary:** Collection of TouchDesigner tools and examples by L05. Includes `.tox` files created for teaching and answering community questions on Facebook/TouchDesigner forums. Mix of older and newer examples covering various TD concepts.
- **Last updated:** Oct 2019

---

## 4. monkeymonk/awesome-touchdesigner

- **Stars:** 149
- **URL:** https://github.com/monkeymonk/awesome-touchdesigner
- **Python focus:** Yes (includes Python cheat sheets and TD Python resources)
- **Summary:** The most comprehensive curated list of TouchDesigner resources on GitHub. Covers official links, tutorials & YouTube channels (Interactive & Immersive HQ, elekktronaut, Matthew Ragan, exsstas), cheat sheets (incl. Python), plugins & tools (MediaPipe, RayTK, TDAbleton, GeoPix, TouchDiffusion, TDNeuron, TouchPy, etc.), assets & marketplaces, generative art inspiration, community (Forum, Discord, Slack, Reddit), events, artists & studios, and other VJ software. Regularly updated with 15 commits and a link-checking script.
- **Last updated:** Feb 2026

---

## 5. ok-vj/okVJ-Shared

- **Stars:** 67
- **URL:** https://github.com/ok-vj/okVJ-Shared
- **Python focus:** Yes (Python expression-based control system)
- **Summary:** A modular VJ control surface for TouchDesigner. Features a UI panel with knobs, sliders, buttons, XY pads, step sequencer, XY modulator, audio analysis, palette generator, and preset management (Tau Ceti engine). Supports MIDI mapping, drag-and-drop parameter binding, edit mode for layout building, and automatic component mappings. Designed for live performance workflows.
- **Last updated:** Active (Jun 2026)

---

## 6. Richard-Burns/Palette-Tools

- **Stars:** 34
- **URL:** https://github.com/Richard-Burns/Palette-Tools
- **Python focus:** Some (Python-backed UI and logic in certain components)
- **Summary:** Collection of palette-ready TouchDesigner tools: BlockGlitch, BrokenVHS (Shadertoy port), CameraFrustrum, DroneCamera (joystick-controlled), FilmFilter, LineGarbler, LTCEmulator, PerspectiveToOrthographic, PointCloudClipper, PointCloudToPLY, SimpleBloom, TextGarbler, TracelineMaker, and UIKit. Created by the author of SimpleMixer. Practical, performance-conscious components.
- **Last updated:** Feb 2026

---

## 7. riebschlager/touchdesigner-playground

- **Stars:** 90
- **URL:** https://github.com/riebschlager/touchdesigner-playground
- **Python focus:** Minimal (mostly SOP/TOP networks with some Python)
- **Summary:** A learning journal of TouchDesigner sketches — 30+ `.toe` files covering instancing, noise-driven geometry, particle systems, shaders, feedback loops, and API data fetching. Each has a thumbnail preview. Ranges from simple (timer-switch, gradient-circles) to advanced (brush-ribbon 3D instancing, face-mixing collage). Not production-optimized but great for learning techniques.
- **Last updated:** Dec 2020

---

## 8. TouchDesigner/CustomOperatorSamples

- **Stars:** 120
- **URL:** https://github.com/TouchDesigner/CustomOperatorSamples
- **Python focus:** No (C++)
- **Summary:** Official Derivative repo with C++ custom operator examples including TOPs (CannyEdge, Cuda, ObjectDetector, OpticalFlow), CHOPs (OneEuro, TimeSlice), SOPs (Spiral, Sprinkle, IntersectPoints), and DATs. Includes comprehensive build guides for Windows (VS) and macOS (CMake) plus OpenCV/CUDA integration docs.
- **Last updated:** Mar 2026

---

## 9. TouchDesigner/GenerativeDesign

- **Stars:** 190
- **URL:** https://github.com/TouchDesigner/GenerativeDesign
- **Python focus:** Yes (the examples use Python heavily)
- **Summary:** Official Derivative repo porting examples from the book *Generative Design* into TouchDesigner. Covers procedural patterns, color systems, form, and data-driven visuals. A great learning resource for generative art within TD.
- **Last updated:** 2019 (archived)
