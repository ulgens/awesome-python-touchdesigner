# Richard-Burns/Palette-Tools

**Stars:** 34 | **URL:** https://github.com/Richard-Burns/Palette-Tools | **Language:** Python/TD

## Overview

A collection of TouchDesigner palette-ready tools by Richard Burns (creator of SimpleMixer).

## Components

| Component | Description |
|-----------|-------------|
| **BlockGlitch** | Row/column block glitch filter with per-channel control. Stackable. |
| **BrokenVHS** | VHS shader effect (Shadertoy port, commercial-use cleared). |
| **CameraFrustrum** | Camera with optional frustum display. |
| **DroneCamera** | Joystick-controlled camera (L/R triggers for up/down). |
| **FilmFilter** | Notch-style film filter with chromatic aberration + luma blur. |
| **LineGarbler** | Row/column displacement glitch. Thresholdable for targeted effect. |
| **LTCEmulator** | Custom timeline-to-LTC timecode converter with offset support. |
| **PerspectiveToOrthographic** | SOP remapping for 3D laser effects. |
| **PointCloudClipper** | 32-bit RGB key for point cloud clipping outside 0-1 range. |
| **PointCloudToPLY** | XYZ+RGB TOP to PLY file exporter. |
| **SimpleBloom** | Lightweight fake bloom. |
| **TextGarbler** | Animated hacker/Matrix-style text garble with typing cursor. |
| **TracelineMaker** | SOP trace line creator with animation export. |
| **UIKit** | Lightweight UI kit (used in SimpleMixer 2023). |

## Python Usage

Some components use Python for UI logic, parameter binding, and file I/O (PointCloudToPLY, LTCEmulator).

## Relevance

Practical, performance-conscious tools for VJ and production work. Actively maintained (Feb 2026).
