# TouchDesigner/GenerativeDesign

**Stars:** 190 | **URL:** https://github.com/TouchDesigner/GenerativeDesign | **Language:** Python

## Overview

Official Derivative repo porting examples from the book *Generative Design* by Benedikt Gross, Julia Laub, and Claudius Lazzeroni into TouchDesigner.

## What's Inside

Procedural pattern generation, color system exploration, form-finding algorithms, data-driven visualizations, and interactive design techniques. Each example demonstrates a generative design concept translated to TD's node-based paradigm.

## Python Usage

The examples use Python extensively for parameter control, randomization, data binding, and generative logic.

## Relevance

Excellent learning resource for artists and designers moving into generative art with TouchDesigner. Last updated 2019 but the concepts remain highly relevant.
