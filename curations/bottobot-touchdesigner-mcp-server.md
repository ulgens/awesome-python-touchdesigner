# bottobot/touchdesigner-mcp-server

**Stars:** 67 | **URL:** https://github.com/bottobot/touchdesigner-mcp-server | **Language:** JavaScript (Node.js)

## Overview

Comprehensive TouchDesigner Documentation MCP Server. Bridges TD operator docs, Python API references, and tutorials to AI coding assistants (Claude, GPT). Zero-config for VS Code/Codium.

## What's Inside

- **661 operators** — Full docs across all 7 families (CHOP 170, TOP 147, SOP 113, POP 102, DAT 75, COMP 41, MAT 13)
- **14 interactive tutorials** — GLSL, C++, CUDA, Shared Memory, Video Streaming, TDBitwig
- **214 Python API classes** — 1,674+ methods with version compatibility annotation
- **21 MCP tools** — Operator docs, search, comparison, workflow suggestions, tutorials, Python API, version system, experimental techniques, GLSL patterns, network templates
- **32 workflow patterns** — Curated operator chain patterns with 72 transitions
- **Version system** — All stable TD releases (099→2025) + experimental build series (2025.30000, 2021.30000)

## Python Usage

Not Python itself, but serves TD Python API documentation. Key tools: `get_python_api` (class docs), `search_python_api` (method/member search), `list_python_classes` (browse by category). Version-filtered Python API queries.

## Relevance

The most comprehensive TD documentation MCP server available. Covers the full TD Python API surface with version-aware search. npm package: `@bottobot/td-mcp`.
