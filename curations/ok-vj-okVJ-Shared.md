# ok-vj/okVJ-Shared

**Stars:** 67 | **URL:** https://github.com/ok-vj/okVJ-Shared | **Language:** Python

## Overview

A modular VJ control surface and component library for live performance in TouchDesigner. Designed as a replacement for traditional VJ hardware in software.

## Core System: okVJ_UI

A drag-and-drop modular control surface with:
- **Edit Mode** — build custom layouts, add/remove/resize components
- **MIDI Map Mode** — map any control to MIDI hardware
- **Preset Manager** — save/recall panel states (Tau Ceti engine)
- **Automatic Mappings** — components auto-wire to compatible peers

## Components

- **Knobs, Sliders, Buttons** — parameter banks with toggle/momentary modes
- **XY Pad** — dual-axis output, dual MIDI mapping
- **Radio Buttons** — exclusive selection, DAT-table driven labels
- **Audio Analysis** — 4-band (low/mid/high/total) with filter tuning
- **Step Sequencer** — 4 sequences, BPM or audio-trigger clocking
- **XY Modulator** — sin/cosine, noise, S&H, manual modes
- **Palette Generator** — color palette with harmony modes
- **Tap Tempo** — global BPM tapping
- **topViewer** — TOP preview panel

## Python Usage

Heavily Python-driven: parameter expressions, MIDI bindings, preset serialization, component automapping logic.

## Relevance

Active development. Ideal for VJs and live performers building custom TD performance rigs.
